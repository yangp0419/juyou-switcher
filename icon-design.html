如何使用这些图标

  1. 点击你喜欢的图标下方的「下载此图标」按钮，会下载一个 HTML 文件
  2. 在浏览器中打开下载的 HTML 文件，对图标进行截图（建议使用 macOS 自带截图工具）
  3. 或者使用 pnpm tauri icon your-icon.png 命令自动生成所有尺寸的图标
  4. 生成的图标会自动替换到 src-tauri/icons/ 目录
  5. 重新运行 pnpm tauri build 构建应用即可看到新图标
