## 点击流程

### 流程 1：成功导入

```
1. 点击 "一键配置" 按钮
   ↓
2. 显示加载提示
   🔄 正在导入 CODEX 配置...
   ↓
3. 后端读取配置文件
   ~/.codex/settings.json + auth.json
   ↓
4. 保存到数据库
   创建 "default" Provider
   ↓
5. 显示成功提示
   ✅ 配置导入成功！已自动配置 CODEX 供应商
   ↓
6. 1.5秒后自动跳转
   → 设置页面
```

### 流程 2：配置已存在

```
1. 点击 "一键配置" 按钮
   ↓
2. 检测到数据库中已有配置
   ↓
3. 显示提示
   ℹ️ 配置已存在，无需重复导入
   ↓
4. 不执行任何操作
```

### 流程 3：未登录

```
1. 点击 "一键配置" 按钮
   ↓
2. 检测到未登录
   ↓
3. 弹出登录框
   ┌────────────────────┐
   │  用户登录          │
   │  ┌──────────────┐  │
   │  │ 用户名       │  │
   │  ┌──────────────┐  │
   │  │ 密码         │  │
   │  [登录]            │
   └────────────────────┘
   ↓
4. 登录成功后继续导入
```

### 流程 4：配置文件不存在

```
1. 点击 "一键配置" 按钮
   ↓
2. 后端尝试读取配置文件
   ~/.codex/settings.json
   ↓
3. 文件不存在
   ↓
4. 显示错误提示
   ❌ Codex 配置文件不存在
      请先使用 Codex 创建配置
```

## 实际代码片段

```tsx
// Agent 选择器
<div className="grid grid-cols-3 gap-3">
  {[
    { id: "claude", name: "Claude", icon: "🤖", color: "bg-purple-50" },
    { id: "codex", name: "Codex", icon: "💻", color: "bg-blue-50" },
    { id: "gemini", name: "Gemini", icon: "✨", color: "bg-green-50" },
  ].map((agent) => (
    <button
      key={agent.id}
      onClick={() => setSelectedAgent(agent.id)}
      className={cn(
        "flex flex-col items-center justify-center gap-2 rounded-lg border-2 px-4 py-3 transition-all",
        agent.color,
        selectedAgent === agent.id
          ? "ring-2 ring-[#0b65d8] ring-offset-2"
          : "opacity-60"
      )}
    >
      <span className="text-2xl">{agent.icon}</span>
      <span className="text-xs font-semibold">{agent.name}</span>
    </button>
  ))}
</div>
```
